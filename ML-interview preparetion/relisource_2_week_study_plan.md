# ReliSource Junior ML Engineer - 2 Week Study Plan

This plan assumes 2 to 3 hours per day. Adjust as needed. Each day has one main topic and one practice task.

## Week 1

**Day 1: Fundamentals Refresh**
- Review bias-variance, overfitting, regularization.
- Practice: write 10 short answers from the written exam guide.

**Day 2: Metrics and Evaluation**
- Review precision, recall, F1, ROC-AUC, PR-AUC, calibration.
- Practice: compute metrics by hand for two confusion matrices.

**Day 3: Classical ML Models**
- Review logistic regression, SVM, decision trees, random forest, boosting.
- Practice: explain when to choose each model (short paragraph each).

**Day 4: Feature Engineering**
- Review text features (TF-IDF, n-grams, embeddings) and tabular features.
- Practice: design features for a fake dataset and justify them.

**Day 5: Data Leakage + Splits**
- Review leakage examples and correct splitting strategies.
- Practice: design a correct pipeline for time-series data.

**Day 6: MLOps Basics**
- Review pipelines, model registry, monitoring, drift.
- Practice: draw a simple pipeline diagram and explain each step.

**Day 7: Mock Exam Part 1**
- Take the mock exam (Sections A and B only).
- Review mistakes and re-study weak areas.

## Week 2

**Day 8: Deep Learning Basics**
- Review CNNs, RNNs, Transformers, transfer learning.
- Practice: explain which model fits which task and why.

**Day 9: Applied Case Questions**
- Review case studies (churn, fraud, sentiment).
- Practice: outline an end-to-end solution in 10 bullet points.

**Day 10: MLOps + Azure ML**
- Review Azure ML pipelines and endpoints, basic cloud concepts.
- Practice: describe a deployment plan in 5 steps.

**Day 11: Coding Practice**
- Practice data cleaning, metrics coding, and a small pipeline.
- Timebox each task to 30 to 40 minutes.

**Day 12: Mock Interview Practice**
- Run the mock interview script out loud.
- Record and improve your explanation clarity.

**Day 13: Mock Exam Part 2**
- Take the mock exam (Sections C and D only).
- Review mistakes and write model answers.

**Day 14: Final Review**
- Revisit all weak topics.
- Write a 1-page quick revision sheet.
